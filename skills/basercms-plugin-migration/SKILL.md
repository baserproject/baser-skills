# 【非推奨】basercms-plugin-migration はリネームされました

このスキルは **`basercms-plugin-5x-update`** にリネームされました。

- baserCMS プラグインの **5.2系 → 5.3系（同一メジャー内のマイナーアップデート）** の移行レシピ
  → **`basercms-plugin-5x-update`** を参照してください。
- baserCMS **4 → 5（メジャーアップグレード、CakePHP 2→5 の全面書き換え）**
  → **`basercms-plugin-4-to-5-upgrade`** を参照してください。

> 命名規約: マイナーバージョンアップ＝`update`、メジャーバージョンアップ＝`upgrade`。
> このスタブは旧名（`basercms-plugin-migration`）での参照互換のために一定期間だけ残置します。内容は新スキルに移管済みです。新しい知見の追記や参照は新名に対して行ってください。
